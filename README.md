### Array Copy and Reverse

Copies an array and creates a reversed version of it.  

**Description:**  
This program copies the original array into a new array and also creates another array with elements in reverse order.  
It demonstrates array copying and reversing techniques in Java.

**Example:**  
Input → `[1,2,3,4,5]`  
Output →  
ArrayA: [1, 2, 3, 4, 5]  
----------------  
ArrayB: [1, 2, 3, 4, 5]  
----------------  
ArrayC: [5, 4, 3, 2, 1]
